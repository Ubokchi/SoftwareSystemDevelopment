GET /article/1: Java 객체(Article) → @ResponseBody(JSON data) (ArticleController#getInJson())

GET /articles: Java 객체(List<Article>) → @ResponseBody(JSON data) (ArticleController#getArticlesInJson())

POST /articles (Ajax call in postJson()): @RequestBody(JSON data) → Java 객체(List<Article>) → 내용 수정 → @ResponseBody(JSON data) (ArticleController#postArticlesInJson())

Article form: Form 입력 → Ajax call(POST) → @RequestBody(JSON data) → Java 객체(List<Article>) → 내용 수정 → @ResponseBody(JSON data) (ArticleController#postArticlesInJson())

GET /messages: Java 객체(MessageList) → @ResponseBody(XML data) (MessageController.getMsgsInXml())

POST /messages (Ajax call in postXml()): @RequestBody(XML data) → Java 객체(MessageList) → 내용 수정 → @ResponseBody(XML data) (MessageController#postMsgsInXml())

GET /category/101, GET /category/102: Java 객체(Category) → @ResponseBody(JSON data) (CategoryController#getCategory())

GET /category/101/product/PT: Java 객체(Product) → @ResponseBody(JSON data) (CategoryController#getProduct())
